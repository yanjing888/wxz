# 用牛顿环测量平凸透镜曲率半径：实验报告整理版

## 实验目的

1. 观察牛顿环等厚干涉现象，理解明暗环与空气膜厚度之间的关系。
2. 掌握利用牛顿环测量平凸透镜曲率半径的方法。
3. 学习用逐差法处理干涉环直径数据，减小中心定位误差。

## 实验原理

平凸透镜与平面玻璃板接触后形成空气薄膜。单色光照射时，空气膜上下表面反射光发生干涉，形成同心圆环。对反射暗环有：

```text
D_m^2 = 4mRλ
```

其中 `D_m` 为第 `m` 级暗环直径，`R` 为透镜曲率半径，`λ` 为光波波长。相隔 `k` 级的两暗环可用逐差法计算：

```text
R = (D_{m+k}^2 - D_m^2) / (4kλ)
```

## 实验步骤

1. 调节钠光灯和读数显微镜，使牛顿环条纹清晰。
2. 选定同一直径方向，分别读取若干级暗环的左右位置。
3. 计算每一级暗环直径和直径平方。
4. 用逐差法求出多组 `R`，取平均值。

## 误差分析

主要误差来自显微镜读数、条纹边界判断、中心环偏移、透镜和平玻璃接触不良以及光源准直不充分。使用逐差法可以削弱中心定位误差，但无法完全消除读数和仪器误差。

## 结论

牛顿环实验验证了等厚干涉规律。通过测量不同级次暗环直径并进行逐差处理，可求得平凸透镜曲率半径。
