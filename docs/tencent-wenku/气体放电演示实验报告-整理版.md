# 气体放电演示实验报告整理版

## 一、演示目的

1. 观察气体放电的几种形式，如电晕放电、火花放电和电弧放电。
2. 理解电场强度、电极形状和气体击穿之间的关系。
3. 认识尖端放电现象及其应用。

## 二、实验原理

在强电场作用下，气体分子被电离，形成带电粒子并产生放电现象。尖端电极附近电场更强，因此更容易发生电晕放电或火花放电。

击穿条件与电压、电极间距、气体压强和电极形状有关。

## 三、演示步骤

1. 将尖端电极和球形电极分别与高压电源连接。
2. 调整电极与平板电极之间距离。
3. 缓慢升高电压，观察不同电极处的放电现象。
4. 比较尖端电极和球形电极的放电难易程度。

## 五、误差与安全

高压演示必须由教师操作或在教师监督下进行。观察者应保持安全距离，不触碰电极和导线。环境湿度和电极间距会影响放电现象。

## 六、结论

气体放电现象说明空气在强电场下可被击穿。尖端处电场集中，更容易产生放电，这也是避雷针等装置的物理基础。
