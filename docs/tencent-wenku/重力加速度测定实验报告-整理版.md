# 重力加速度的测定：实验报告整理版

> 状态：页面可见预览提到“重力加速度的测定”。本文为实验报告整理版。

## 一、实验任务

测定当地重力加速度 `g`，并分析测量方法和误差来源。

## 二、实验目的

1. 掌握利用自由落体、单摆或打点计时器测量重力加速度的基本方法。
2. 学习对多次测量数据进行平均和误差分析。
3. 理解物理模型假设对测量结果的影响。

## 三、实验原理

以单摆法为例，单摆周期满足：

```text
T = 2π√(l/g)
```

因此：

```text
g = 4π²l / T²
```

其中 `l` 为摆长，`T` 为单摆周期。

## 四、实验步骤

1. 测量摆长，包括悬点到摆球球心的距离。
2. 使摆角较小，释放摆球并计时多个周期。
3. 计算平均周期 `T`。
4. 代入公式求 `g`。

## 六、误差分析

摆长测量、秒表反应时间、摆角过大、空气阻力和支点摩擦都会影响结果。计时多个周期并取平均可降低反应时间误差。

## 七、结论

实验可得到当地重力加速度的近似值。若相对误差较大，应重点检查摆长定义、计时方法和小角度条件是否满足。
