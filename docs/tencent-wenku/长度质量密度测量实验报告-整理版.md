# 长度、质量、密度的测量：实验报告整理版

> 来源：https://wenku.docs.qq.com/detail?docId=r6ry5y358i&source=related

## 一、实验目的

1. 掌握游标卡尺和螺旋测微计的读数方法。
2. 学会使用天平测量质量。
3. 通过长度和质量测量计算规则物体密度。
4. 练习有效数字和误差分析。

## 二、实验仪器

游标卡尺、螺旋测微计、物理天平或电子天平、待测圆柱体或小球。

## 三、实验原理

圆柱体密度：

```text
ρ = m / V
V = πd²h / 4
```

其中 `m` 为质量，`d` 为直径，`h` 为高度。

## 四、实验步骤

1. 检查游标卡尺和螺旋测微计零点误差。
2. 多次测量物体直径和高度。
3. 使用天平测量物体质量。
4. 计算体积和密度。
5. 分析测量误差。

## 六、误差分析

主要误差包括零点误差、读数视差、物体形状不规则、天平灵敏度和有效数字保留不当。多次测量并取平均能降低偶然误差。

## 七、结论

通过长度和质量测量可计算待测物体密度，并理解间接测量中误差传递的基本思想。
