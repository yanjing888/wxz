# 热敏电阻特性研究：实验报告整理版

> 来源：https://wenku.docs.qq.com/detail?docId=I02OJrpAQa&source=related  
> 状态：页面可见预览提到热敏电阻及电桥法。本文为实验报告整理版。

## 一、实验目的

1. 了解热敏电阻阻值随温度变化的特性。
2. 掌握电桥法测量电阻的基本方法。
3. 绘制热敏电阻 `R-T` 特性曲线，并分析其变化规律。

## 二、实验仪器

热敏电阻、恒温水浴、电桥或数字万用表、温度计、直流电源、导线。

## 三、实验原理

热敏电阻是阻值对温度敏感的半导体元件。常见负温度系数热敏电阻的阻值随温度升高而减小。实验中可在不同温度下测量电阻值，并绘制曲线。

电桥平衡时有：

```text
R_x / R_2 = R_1 / R_3
```

从而可求待测电阻 `R_x`。

## 四、实验步骤

1. 将热敏电阻放入水浴中，缓慢升温。
2. 每隔一定温度记录一次电阻值。
3. 保持温度稳定后再读数。
4. 绘制 `R-T` 曲线或 `ln R - 1/T` 曲线。

## 六、误差分析

温度未稳定、热敏电阻自热、电桥平衡判断和温度计读数都会造成误差。实验时应控制电流较小，并等待热平衡后读数。

## 七、结论

热敏电阻阻值随温度变化显著，可用于温度测量和自动控制。通过曲线拟合可进一步得到其温度系数。
