## 实验目的与要求

实验目的应说明本实验要观察的物理现象、要掌握的测量方法以及最终要得到的物理量。实验要求应包含操作规范、数据精度和报告提交要求。

## 实验原理

写出核心公式，并说明各符号含义。若需要作图或拟合，应说明坐标变量和斜率含义。

## 实验内容与步骤

1. 检查仪器并记录零点。
2. 按照实验装置图连接或调整。
3. 进行多次测量。
4. 记录原始数据。
5. 完成计算和误差分析。

## 数据处理示例

```text
平均值 = 多次测量结果之和 / 测量次数
相对误差 = |测量值 - 理论值| / 理论值 × 100%
```

## 结论写法

结论应包含测量结果、误差范围和主要误差来源。避免只写“符合理论”。
