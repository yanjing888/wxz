# 大学物理实验报告：用 CCD 成像系统观测牛顿环

> 来源链接：https://wenku.docs.qq.com/detail?docId=NaF1QvNGNR  
> 抓取说明：页面标题和部分预览内容可公开查看，页面同时显示“查看剩余全文”。本报告为依据可见信息与牛顿环实验原理重新整理的实验报告，不复制原文全文。表中数据为示例数据，可替换为实际测量值。

## 一、实验名称

用 CCD 成像系统观测牛顿环并测量平凸透镜曲率半径

## 二、实验目的

1. 观察空气薄膜产生的等厚干涉现象，理解牛顿环明暗条纹的形成原因。
2. 熟悉 CCD 成像系统、读数软件或图像测量工具在光学实验中的使用。
3. 通过测量牛顿环直径，掌握利用干涉条纹计算平凸透镜曲率半径的方法。
4. 练习用逐差法处理干涉环数据，减小中心定位误差与单次读数误差。

## 三、实验仪器

钠光灯、扩束及准直装置、牛顿环仪、平凸透镜、平面玻璃板、半透半反镜或透射观察装置、CCD 摄像头、计算机图像采集系统、图像测量软件。

## 四、实验原理

平凸透镜放在平面玻璃板上时，两者之间形成厚度随半径逐渐增大的空气薄膜。单色光通过该空气膜时，在薄膜上下表面形成的透射光相干叠加，产生同心圆形干涉条纹。由于空气膜厚度只与离接触点的距离有关，所以等厚线表现为一系列圆环。

设平凸透镜曲率半径为 `R`，第 `m` 级透射亮环半径为 `r_m`，入射单色光波长为 `λ`。在近轴条件下，空气膜厚度满足：

```text
e ≈ r_m^2 / (2R)
```

透射光观察时，中心附近通常为亮斑。第 `m` 级亮环直径 `D_m = 2r_m` 满足近似关系：

```text
D_m^2 = 4mRλ
```

为避免直接判断中心级次，可采用逐差法：

```text
R = (D_{m+k}^2 - D_m^2) / (4kλ)
```

本整理版取钠黄光波长：

```text
λ = 589.3 nm = 5.893 × 10^-7 m
```

## 五、实验步骤

1. 打开钠光灯并预热，使光源稳定。
2. 调节光路，使入射光经过准直后照射到牛顿环仪上。
3. 调整 CCD 摄像头焦距和曝光参数，使牛顿环图像清晰、对比度适中。
4. 在图像采集软件中确定圆环中心，沿同一直径方向测量若干级亮环的左、右位置。
5. 由左右位置差得到各级亮环直径，并换算为实际长度。
6. 采用逐差法计算平凸透镜曲率半径，记录平均值并进行误差分析。

## 七、数据处理

取 `k = 5`，计算相隔 5 级的直径平方差：

```text
R = (D_{m+5}^2 - D_m^2) / (20λ)
```

其中 `D^2` 需由 `mm^2` 换算为 `m^2`。

| 数据组 | D_{m+5}^2 - D_m^2 / mm^2 | R / m |
|---:|---:|---:|
| 10 - 5  | 10.277 | 0.872 |
| 11 - 6  | 10.338 | 0.877 |

| 13 - 8  | 10.334 | 0.877 |
| 14 - 9  | 10.265 | 0.871 |
| 15 - 10 | 10.286 | 0.873 |

平均曲率半径：

```text
R_avg = 0.874 m
```

## 八、误差分析

1. CCD 图像中条纹边界存在灰度过渡，选取环线位置时会产生读数误差。
2. 若光路未严格准直或牛顿环仪未水平放置，环纹可能偏心，导致左右读数不对称。
3. 平凸透镜与平面玻璃板之间若有灰尘，会破坏局部空气膜厚度，影响条纹清晰度。
4. 中心点定位不准会影响各级环半径，但逐差法能削弱这一系统影响。
5. 软件像素标定或长度换算存在误差，也会传递到曲率半径结果中。

## 九、实验结论

实验中观察到了清晰的牛顿环等厚干涉图样。根据透射亮环直径平方与级次的线性关系，采用逐差法计算得到平凸透镜曲率半径约为：

```text
R = 0.874 m
```

结果说明，CCD 成像系统能够较方便地记录和测量牛顿环图样；使用逐差法处理数据可以提高曲率半径测量的稳定性。

## 十、注意事项

1. 采集图像时应避免曝光过强，否则中心和低级条纹容易饱和。
2. 测量时应选取同一直径方向，避免因椭圆化或偏心造成额外误差。
3. 透镜和玻璃板表面应保持清洁，接触时避免用力压迫。
4. 若使用反射光观察，中心明暗和条纹条件与透射光不同，应按对应公式处理。
