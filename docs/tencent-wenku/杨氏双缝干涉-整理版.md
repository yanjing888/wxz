# 大学物理：杨氏双缝干涉整理版

> 来源：https://wenku.docs.qq.com/detail?docId=heiDPlI30C&docType=4&source=related  
> 状态：页面可见目录式预览。本文为学习笔记式整理。

## 一、实验目的

1. 观察杨氏双缝干涉条纹。
2. 理解相干光源、光程差和干涉明暗条件。
3. 掌握利用条纹间距测量光波波长的方法。

## 二、实验原理

单色光通过双缝后，两缝可近似看作两个相干光源。在屏幕上任一点，两束光的光程差为：

```text
δ = d sinθ
```

小角度条件下：

```text
sinθ ≈ x / L
δ ≈ dx / L
```

明纹条件：

```text
δ = kλ
```

暗纹条件：

```text
δ = (k + 1/2)λ
```

相邻明纹间距为：

```text
Δx = Lλ / d
```

## 三、实验步骤

1. 调整光源、单缝、双缝和观察屏，使条纹清晰。
2. 测量双缝到屏的距离 `L`。
3. 记录若干条明纹的位置，计算平均条纹间距 `Δx`。
4. 由 `λ = dΔx / L` 计算光波波长。

## 五、误差分析

条纹宽度、屏幕读数、光源单色性、双缝间距误差和光路不共轴都会影响波长计算。可通过测量多级条纹间距并取平均来减小偶然误差。

## 六、结论

杨氏双缝干涉实验展示了光的波动性。条纹间距与波长成正比，与双缝间距成反比。
