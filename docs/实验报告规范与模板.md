# 物小智 · 实验总结报告规范

> 版本：v2.0（已实现当前可用内容）  
> 页面预览与 DOCX 下载共用 `buildReportData()` 数据，模块顺序与字段一致。

---

## 一、当前报告包含什么

本阶段报告覆盖**平台已有数据**，暂不包含教研配置的「实验目的 / 原理 / 仪器 / 分析结论」等正文（见文末「后续扩展」）。

| 模块 | 内容 | 数据来源 |
|------|------|----------|
| 封面 | 实验名称、姓名、班级、生成时间、会话编号 | 会话 + 生成时刻 |
| 1. 实验步骤回顾 | 各步标题与说明 | `experiments/*.json` → `steps` |
| 2. 实操回顾与数据统计 | 纠错次数、标注点、教程查阅、L3 告警 | 会话统计字段 |
| 3. 操作纠错记录 | 步骤、类型、时间、问题描述 | `correction_logs` |
| 4. 实验数据记录 | 步骤、提交时间、采集值、校验结果 | `session_data_logs` |
| 5. 环境安全巡检记录 | 时间、等级、依据、建议、是否抽帧 | `env_check_logs` |
| 6. 知识巩固建议 | 项目符号列表 | `reportKnowledge` |
| 7. 后续学习路径 | 编号列表 | `reportPath` |

## 二、模块顺序（页面 = DOCX）

```
实验总结报告
├── 封面信息
├── 1. 实验步骤回顾
├── 2. 实操回顾与数据统计
├── 3. 操作纠错记录
├── 4. 实验数据记录
├── 5. 环境安全巡检记录
├── 6. 知识巩固建议
└── 7. 后续学习路径
```

## 三、字段说明

### 封面

| 字段 | 说明 |
|------|------|
| experimentName | 实验名称 |
| studentName | 学生姓名 |
| studentClass | 班级 |
| generatedAt | 生成时间（`yyyy年MM月dd日 HH:mm:ss`） |
| sessionId | 会话编号 |

### stepSummaries[]

| 字段 | 说明 |
|------|------|
| stepNo | 步骤序号 |
| title | 步骤标题 |
| desc | 步骤说明（来自 JSON `steps.*.desc`） |

### dataLogEntries[]

| 字段 | 说明 |
|------|------|
| stepTitle | 所属步骤 |
| submittedAt | 提交时间 |
| valuesSummary | 采集数据摘要（`key=value` 拼接） |
| validationSummary | `通过` / 错误信息 / `—` |

### corrections[]

页面与 DOCX **仅展示**：stepTitle、errorType、createdAt、detail。  
**不包含** AI 反馈全文（feedback）。

### envLogs[]

| 字段 | DOCX 展示 |
|------|-----------|
| createdAt | 巡检时间 |
| level | L0 正常 / L1 注意 / … |
| summary | 判断依据 |
| suggestion | 处置建议 |
| snapshotUrl | **已采集 / 未采集**（不写 URL） |

## 四、空状态文案（页面与 DOCX 统一）

| 模块 | 无数据时 |
|------|----------|
| 实验步骤回顾 | 不显示该节（无 steps 配置） |
| 操作纠错记录 | 操作表现优异！整个实验过程中未发现明显操作逻辑错误。 |
| 实验数据记录 | 本次实验未提交结构化实验数据。 |
| 环境安全巡检 | 本次实验未产生环境巡检记录（未开启安全监测或未触发巡检）。 |
| 知识巩固 / 学习路径 | 暂无内容。 |

---

## 五、实验数据如何进入报告

学生在步骤中**提交数据**（仪器采集或手工录入）后写入 `session_data_logs`，生成报告时汇总为 **§4 实验数据记录**。

- 不是 AI 自动猜数，而是**已提交的结构化数据**  
- 未提交的步骤不会出现在数据表中

---

## 六、DOCX 版式（当前实现）

- 标题居中，二级节标题加粗 14pt  
- 封面、统计、纠错、数据、巡检：**表格呈现**  
- 知识建议：项目符号；学习路径：编号列表  
- 页末说明：「本报告由物小智智能实验平台根据本次实验过程数据自动生成。」
