# 牛顿环实验教学知识库

本文档用于 Dify 的“实验教学知识库”。它面向原理讲解、步骤指导、数据处理、报告生成和学习建议；不承担视觉判错规则职责。视觉纠错规则请使用 `docs/visual-rules/newton_rings.md`。

## 一、实验目的

1. 观察等厚干涉现象，理解牛顿环的形成条件。
2. 掌握读数显微镜观察和测量干涉环直径的方法。
3. 学会利用暗环直径与环序关系计算平凸透镜曲率半径或单色光波长。
4. 训练单向逼近读数、重复测量、单位换算和误差分析能力。

## 二、实验原理

牛顿环是由平凸透镜和平玻璃板之间形成的空气薄膜产生的等厚干涉现象。当单色光垂直入射时，空气膜上下表面的反射光发生干涉，形成一系列以接触点为中心的同心明暗圆环。

对反射光暗环，常用关系为：

```text
r_m^2 = m λ R
D_m^2 = 4 m λ R
```

其中：

- `r_m`：第 m 级暗环半径；
- `D_m`：第 m 级暗环直径；
- `λ`：单色光波长；
- `R`：平凸透镜曲率半径；
- `m`：暗环级数。

为了减小环心定位误差，实验中常使用两条暗环直径平方差：

```text
D_m^2 - D_n^2 = 4 (m - n) λ R
```

若已知波长 `λ`，可求曲率半径：

```text
R = (D_m^2 - D_n^2) / [4 (m - n) λ]
```

若已知曲率半径 `R`，可求波长：

```text
λ = (D_m^2 - D_n^2) / [4 (m - n) R]
```

## 三、仪器与材料

常见仪器包括：

- 牛顿环仪；
- 平凸透镜和平玻璃板；
- 读数显微镜；
- 钠光灯或其他单色光源；
- 反射镜或半透半反镜；
- 数据记录表或采集终端。

## 四、实验步骤

### Step 1：仪器检查与光路调节

目标：检查钠光灯或单色光源、读数显微镜与牛顿环仪支架；调节反射镜与显微镜焦距，使视场中出现清晰、均匀的干涉环纹。

操作要点：

1. 确认光源预热充分，亮度稳定。
2. 调节反射镜，使光束进入显微镜观察光路。
3. 先调清十字丝，再调清牛顿环环纹。
4. 检查透镜和平玻璃接触区域是否清洁。
5. 避免直视强光源。

### Step 2：零级环心定位与视场居中

目标：在显微镜视场中找到干涉中心，使环心接近十字丝交点，并记录初读数 `D0`。

操作要点：

1. 通过载物台粗调，使环心位于视场中央。
2. 微调显微镜，使十字丝交点接近零级暗斑或环心。
3. 记录鼓轮初读数 `D0`。
4. 测量过程中保持鼓轮单向逼近，避免回程误差。

### Step 3：暗环直径测量

目标：从中心向外依次选取暗环，测量同一暗环左右两侧读数，计算暗环直径。

推荐记录字段：

```text
暗环序号 m
左读数 reading_left_mm
右读数 reading_right_mm
直径 D_m = |reading_right_mm - reading_left_mm|
```

操作要点：

1. 选定清晰暗环，例如 m=10、m=15、m=20。
2. 左右读数必须来自同一暗环。
3. 每次读数前确认十字丝与目标暗环对准。
4. 鼓轮保持单向逼近。
5. 建议至少测量 3 组不同环序的数据。

### Step 4：曲率半径或波长计算

目标：根据 `D_m^2 - D_n^2` 与环序差 `Δm` 计算曲率半径或波长。

操作要点：

1. 计算每组暗环直径平方。
2. 计算 `D_m^2 - D_n^2`。
3. 确认 `Δm = m - n`。
4. 统一单位后代入公式。
5. 计算平均值和相对误差。

## 五、数据处理规范

### 直径计算

```text
D_m = |右读数 - 左读数|
```

注意：

- 左右读数必须属于同一暗环；
- 单位通常为 mm；
- 有效数字应与仪器分辨率匹配。

### 平方差计算

```text
D_m^2 - D_n^2
```

注意：

- 若 D 使用 mm，则 D² 单位是 mm²；
- 代入公式前要确认 λ 和 R 的单位与 D² 匹配。

### 常见波长参考

钠光波长约：

```text
λ ≈ 589.3 nm
```

## 六、常见误差来源

1. 环心定位不准导致系统误差。
2. 环纹模糊导致判线不一致。
3. 鼓轮回程误差导致读数偏差。
4. 左右读数选取了不同暗环。
5. 透镜或玻璃表面污染导致环纹畸变。
6. 单位换算错误导致结果数量级异常。
7. 有效数字保留不规范。
